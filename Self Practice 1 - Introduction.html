List Tags

  • ol tag represents ordered list. It can have type as an attribute
  • ul tag represents unordered list
  • li tag lists each of the item in the list

hr tag is used to draw a line inbetween. We can also use attributes like size and noshade


em tag is used to make text italic


strong tag is used to make text bold and emphasizes the text better than b tag

img tag above puts image into the website. src shows source like file or link to image. alt is alternative source when this image cannot be found


The anchor tag defines a hyperlink, which is used to link from one page to another.


Anchor tag also can be used to create a hyperlink to a brand new page
